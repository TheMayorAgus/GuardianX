# GuardianX Landing

Landing page simple para GuardianX, diseñada con Tailwind y animaciones AOS.

Incluye:
- Formulario de contacto vía Netlify
- Modo oscuro toggle
- Animaciones AOS
- Sección de features
- Integración lista para Plausible Analytics (opcional)

### Deploy sugerido
- GitHub Pages, Netlify, Vercel o self-host
